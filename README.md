# 开源机器人学

您好，这是一个帮助机器人学习者入门的开源项目，我会在本项目更新以下内容：

1. 机器人学基础知识点的实现
2. 实用且有趣的算法实现

需要特别指出，本项目的开发要遵循几个原则：

1. **单文件原则**：除了必要的 python 基础模块外，每个实例仅依赖于单个文件的内容
2. **索引原则**：这不是一份教学材料，附带的文档中固然可以简单讲解算法，但更重要的是为初学者指明有价值的参考资料

本项目将持续更新，感谢您的关注~
